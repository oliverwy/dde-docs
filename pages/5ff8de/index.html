# DSettingsDialog的使用

构建一个DSettingsDialog对象以后，首先需要调用void DSettingsDialog::updateSettings(Dtk::Core::DSettings *settings)来指定读取json文件的DSettings，注意只能调用一次。


# DSettingsDialog的国际化

如果是需要国际化该对话框，那么就需要调用另外一个带翻译参数的方法void DSettingsDialog::updateSettings(const QByteArray &translateContext, Core::DSettings *settings)，这个方法同样也只能调用一次。 具体国际化的方法参见《深度应用开发规范》
